Swift - 数组遍历

2016/11/12 posted in  Swift

最近没怎么更新,主要是最近刚入职新公司.正在熟悉环境中… 好了题目是Swift - 数组遍历 主要是项目要使用 Swift 开发,所以需要熟悉下语法.

传统的下标遍历方法

let array = ["张三","李四","王五"]

// 第一种 下标遍历
for i in 0..<array.count {
    print(array[i])
}

for in 遍历元素

// 第二种 for in 遍历元素
for s in array {
    print(s)
}

enum block遍历

// 第三种 enum block 遍历
for e in array.enumerated() {
    print(e)
}

结果:

(0, "张三")
(1, "李四")
(2, "王五")

我们可以看下e的类型–元组

元组方法遍历下标和内容

由上面第三种方法我们可以知道 e 的类型是元组,那么同样我们可以直接使用元组

// 第四种 元组使用
for (n,s) in array.enumerated() {
    print("\(n) \(s)")
}

结果:

0 张三
1 李四
2 王五

反序遍历

// 第五种 反序遍历
for i in array.reversed() {
    print(i)
}

结果

王五
李四
张三

反序遍历 带索引

// 第六种 反序遍历 带索引
for (n,s) in array.enumerated().reversed() {
    print("\(n) \(s)")
}

结果:

2 王五
1 李四
0 张三

注意: 先枚举再反序

总结:Swift 太灵活了…
