Swift - 数组中的增删改与合并

2016/11/12 posted in  Swift

第一篇写了数组遍历的几种情况,这篇主要就是数组中的增删改,也是开发中常用的

OC中数组分为可变(NSMutableArray)和不可变(NSArray) 相应的在Swift中就是 var & let

追加元素

首先定义一个可变数组

var array = ["张三","李四","王五"]

追加一个元素

// 追加元素
array.append("赵六")
print(array)

修改元素

通过下标进行修改

array[0] = "赵四"
print(array)

结果:

["赵四", "李四", "王五", "赵六"]

删除

// 删除元素
array.remove(at: 0)
print(array)

结果:

["赵四", "李四", "王五", "赵六"]
["李四", "王五", "赵六"]

remove 的方法由很多,移除首/尾,移除范围…

array.removeAll(keepingCapacity: true)
print("数组中的元素\(array) 数组容量\(array.capacity)")

结果:

数组中的元素[] 数组容量6

数组合并

var arr0 = [1,2,3]
let arr1 = [3,4,5]
arr0 += arr1
print(arr0)

结果:

[1, 2, 3, 3, 4, 5]

注意: 要合并的数组的类型必须一致,如果不一致会出错. 比如:

var arr0:[Any] = [1,2,3]
let arr1:[Any] = ["2",3,4,5]
arr0 += arr1
print(arr0)
